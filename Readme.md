# Cyber Lab

Laboratório pessoal para documentar meus estudos em cibersegurança.

## Objetivos

- Aprender Linux
- Estudar redes de computadores
- Realizar laboratórios práticos
- Criar projetos para portfólio
- Desenvolver competências para minha primeira oportunidade em TI

## Estrutura

- `anotacoes`: resumos dos assuntos estudados
- `laboratorios`: documentação das atividades práticas
- `scripts`: scripts de automação
- `capturas`: imagens utilizadas na documentação
- `diario-estudos`: registro da evolução nos estudos
